# HTB Sherlock - Meerkat

## Overview

This investigation analyzes network traffic and system logs to determine whether a business management platform has been compromised.

The analysis follows the attacker’s activity from initial access to post-exploitation, providing visibility into how the breach occurred, what actions were performed on the system, and how persistence was established.

Through the examination of PCAP data and alert logs, the investigation reconstructs the full attack chain, including unauthorized access attempts, exploitation of a web application vulnerability, remote command execution, and the establishment of long-term access.

This report highlights the techniques used by the attacker, the impact on the system, and the indicators of compromise that can support detection and response efforts.
