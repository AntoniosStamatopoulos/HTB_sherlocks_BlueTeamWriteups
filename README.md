# HTB Sherlocks – Blue Team Writeups

This repository contains blue team writeups from Hack The Box Sherlock challenges.

The goal is to analyze security incidents using logs and forensic data, reconstruct attack chains, and identify key attacker techniques in enterprise environments.

---

## Scope

The writeups focus on:

- Log analysis (Windows Event Logs, Sysmon)
- Incident response and investigation
- Credential access and lateral movement techniques
- Persistence mechanisms
- Attack chain reconstruction

---

## Structure

Each challenge is organized in a separate folder and includes:

- Investigation summary
- Key findings
- Timeline of events
- Relevant artifacts

---

## Purpose

This repository serves as a portfolio of practical blue team skills and hands-on experience in security analysis.

---

## Disclaimer

All content is based on Hack The Box scenarios and is intended for educational purposes.
